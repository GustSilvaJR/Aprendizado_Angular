Two-Way Data-Biding sincronizando os dados inseridos no input com o valor do atributo na classe.

Dessa forma, o valor default do atributo ja é passado inicialmente para o input, e assim que o input for alterado o valor do atributo também será

Assim impleto duas vias. Se alterar na classe, altera no input. Se alterar no input altera na classe

Valor vindo do atributo: {{twoWayDataBidingInput}}
